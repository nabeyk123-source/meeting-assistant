🎙️ VoiceMeeter Banana 設定ガイド

自分と相手の声を両方文字起こしするための設定です。一度設定すれば次回以降不要です。

Step 1

VoiceMeeter Banana をインストール

公式サイトからダウンロード・インストールvb-audio.com → VoiceMeeter Banana
PCを再起動音声ドライバーの反映に必要です
VoiceMeeter Banana を起動
💡 無料で使えます（寄付歓迎）。企業利用も可能です。
Step 2

Windowsの再生デバイスを変更

サウンドコントロールパネルを開く🔊右クリック →「サウンドの設定」→「サウンドコントロールパネル」
「再生」タブ → VoiceMeeter Input を既定に設定右クリック →「既定のデバイスとして設定」
ℹ️ これでTeams・Zoomの音声がVoiceMeeterに流れます。
Step 3

VoiceMeeter Banana の設定

① Stereo Input 1 にマイクを設定デバイス名をクリック → 物理マイク・ヘッドセットを選択
② Stereo Input 1 の B1 をON（点灯）マイク音声が文字起こし側へ流れます
③ HARDWARE OUT A1 にイヤホンを設定実際に音を聴くデバイスを選択
④ VIRTUAL INPUTS の B1 と A1 をONB1 = 文字起こしへ出力 ／ A1 = イヤホンで聴く
⑤ Menu → System Settings → サンプリングレートを 44100Hz に設定して再起動
✅ イヤホンから音が聞こえれば完了です。
Step 4

Teams / Zoom の音声設定

Teams：設定 → デバイス → スピーカー：VoiceMeeter Input
Teams：設定 → デバイス → マイク：物理マイク
Zoom：設定 → オーディオ → スピーカー：VoiceMeeter Input
Zoom：設定 → オーディオ → マイク：物理マイク
⚠️ イヤホンを変えた場合はA1のデバイスを変更するだけでOKです。
Step 5

動作確認

録音開始 → 「テスト」と話して文字起こし確認
YouTube等を流して相手の声の文字起こし確認
イヤホンから音が正常に聞こえることを確認
❌ うまくいかない場合
→ VIRTUAL INPUTS の B1 が点灯しているか確認
→ Teams のスピーカーが VoiceMeeter Input か確認
→ VoiceMeeter を再起動
🎉 設定完了！録音を開始してください。
